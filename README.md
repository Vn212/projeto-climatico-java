# Projeto Climático em Java 

Projeto simples em Java que consome uma API de clima para exibir
informações climáticas atuais a partir do nome de uma cidade,
utilizando entrada via terminal.

## Funcionalidades
- Entrada de dados pelo usuário (nome da cidade)
- Consumo de API externa de clima
- Tratamento de resposta em JSON
- Exibição de dados climáticos formatados no terminal

## Tecnologias utilizadas
- Java 22
- HttpClient (java.net.http)
- Biblioteca JSON
- API externa de clima

## Como executar o projeto
1. Clone o repositório
2. Crie um arquivo `apikey.txt` na raiz do projeto
3. Insira sua chave da API de clima dentro do arquivo
4. Execute a classe `Climatico`

## Observações
- O arquivo `apikey.txt` não é versionado por segurança
- Projeto com foco em aprendizado de consumo de APIs em Java
